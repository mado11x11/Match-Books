## アプリ名：Match-Books

## 概要
CRUD機能を持つアプリ制作。

紹介したい本の登録・削除・更新、他ユーザーが登録した書籍の閲覧機能。

## 環境
Java 11

Spring Boot 2.6.7

Docker 20.10.12

## 始め方
$ git clone https://github.com/mado11x11/Match-Books.git

$ cd Match-Books

$ docker-compose up -d

$ ./gradlew bootRun
